#/J2EE：J2EE课程作业
课程的8次作业，场景为根据学号/密码查询所选课程成绩，根据结果，返回不同页面。分别用不同的技术实现。
* 该学号/密码不存在；
* 按照要求通过课程，显示课程列表及成绩；
* 有未通过的课程，分别显示通过/未通过的课程列表及成绩，并给出警告信息。<br/>
